Faça um programa que leia um texto informado pelo usuário. Em seguida, calcule o número de caracteres desse texto e exiba uma mensagem conforme abaixo:
Se o número de caracteres for de 0 até 10 O texto: x é PEQUENO, tem y caracteres
Se o número de caracteres for de 11 até 50 O texto: x é MÉDIO, tem y caracteres
Se o número de caracteres for maior que 50 O texto: x é GRANDE, tem y caracteres
